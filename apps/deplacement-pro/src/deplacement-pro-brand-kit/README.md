# Déplacement Pro - Brand System

> Comparez et optimisez vos solutions de déplacements professionnels.

## Quick Start

```bash
# Copy assets to your project
cp -r public/* your-project/public/
cp -r styles/* your-project/src/styles/
cp -r .claude your-project/
cp tailwind.config.ts your-project/
```

### Install fonts
Add these imports to your CSS or `<head>`:
```css
@import url('https://fonts.googleapis.com/css2?family=Urbanist:wght@600;700;800&display=swap');
@import url('https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;600&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Space+Mono:wght@400;500&display=swap');
```
